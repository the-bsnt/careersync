## âš™ï¸ Installation & Setup

Follow these steps to run **CareerSync** locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/the-bsnt/careersync.git
   ```

2. **Move project to XAMPP directory**

   - Copy the cloned project folder.
   - Paste it inside the `htdocs` folder of your XAMPP installation.
     Example:

     ```
     C:\xampp\htdocs\careersync
     ```

3. **Set up the database**

   - Open **phpMyAdmin** by visiting [http://localhost/phpmyadmin](http://localhost/phpmyadmin).
   - Create a new database (e.g., `careersync_db`).
   - Import the provided `.sql` file located inside the project folder (usually `database/careersync_db.sql`).

4. **Start the server**

   - Open **XAMPP Control Panel**.
   - Start **Apache** and **MySQL** services.

5. **Run the project**

   - Open your browser and go to:
     ğŸ‘‰ [http://localhost/careersync](http://localhost/careersync)
